# FastCGI for GT.M

## Description

FastCGI for GT.M is a FastCGI-backend for webservers to deploy web-applications in GT.M
(See also https://github.com/shabiel/fis-gtm and https://fastcgi-archives.github.io/FastCGI_Specification.html)
